## App Details

### Description

Prowl monitors your product pages every day and alerts you when something breaks. From a missing Add to Cart button to a broken variant selector, Prowl catches the issues that silently stop customers from buying — so you can fix them fast.

### Key Features

- Daily automated scans detect issues across your product pages
- AI-powered analysis confirms real problems and filters out false alarms
- Full purchase flow testing verifies customers can actually add to cart
- Screenshot evidence included with every alert so you can see the issue
- Actionable alerts with plain-language explanations and suggested fixes
- Works with any Shopify theme — no setup or code changes required

### How It Works

Add up to 3 product pages to monitor and Prowl takes care of the rest. Every day, Prowl visits your pages like a customer would — checking that images load, prices display, variants work, and the Add to Cart button actually adds to cart. When an issue is found, you get an email with a screenshot, a clear explanation, and a suggested fix.
